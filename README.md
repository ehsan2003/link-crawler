# Link crawler 
crawl links from a website

cli help is enough :) 
```
link-crawler 0.1.0

USAGE:
    link-crawler [OPTIONS] <URL>

ARGS:
    <URL>    The URL to scrape

OPTIONS:
    -h, --help             Print help information
    -r, --regex <REGEX>    regular expression to filter result
    -V, --version          Print version information
```
